Data classification is at the heart of many applications in artificial intelligence, particularly in the medical field. 
In this project, we focused on the classification of electrocardiographic (ECG) signals, a challenge where the accuracy of models plays a crucial role.
By combining supervised and unsupervised approaches, we evaluated the performance of several algorithms on the dataset (such as SVM, decision trees, k-NNs, ...). 
The study also relied on dynamic programming using Dynamic Time Warping (DTW) and on dimensionality reduction techniques such as Principal Component Analysis (PCA).
This report presents our approach, from the preliminary data analysis to the results obtained, including the implementation of the algorithms and the evaluation of their robustness.
